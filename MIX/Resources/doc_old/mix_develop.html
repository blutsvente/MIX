Hallo Leute,
 
ich habe ein paar neue Funktionen in die Globals Klasse eingetragen. Damit kann man seine Perl Packages bekanntmachen, nützlich für's Debugging, um die Versionen zu kennen. Ist noch nicht viel, aber ein Anfang. Später soll man damit noch mehr machen können, z.B. um User-packages und Hook-functions zu registrieren, die dann zu bestimmten Phasen von MIX ausgeführt werden.
 
Beispiele zur Benutzung unten.
 
Gruß,
Thorsten.
 
    # register Perl module with mix
    if (defined $eh) {
        $eh->mix_add_module_info("RegViews", '$Revision: 1.1 $ ', "Utility functions to create different register space views from Reg class object");
    };
 

    
    # retrieve package version
    my $href_info = $eh->mix_get_module_info("RegViews.pm");
    print $href_info->{'version'};
 
 

    # get info about all packages and print it 
    my $href_info = $eh->mix_get_module_info_all();       
    my $dump = Data::Dumper->new([$href_info]);
    print $dump->Dump;
